# 🎬 Netflix Data Analysis Project

## 📌 Project Overview

This project analyzes Netflix data to find trends in content, ratings, and distribution. The goal is to extract business insights and present them using an interactive dashboard.

## 🛠️ Tools Used

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* Power BI
* SQL (basic understanding)

## 🔄 Project Workflow

1. Data Cleaning (handled missing values, removed duplicates)
2. Exploratory Data Analysis (EDA)
3. Data Visualization
4. Dashboard Creation (Power BI)

## 📊 Key Insights

* Netflix has more Movies than TV Shows
* Content production increased rapidly after 2015
* USA is the leading content producer
* TV-MA and TV-14 are the most common ratings
* Drama and International genres dominate
